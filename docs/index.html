Guia passo a passo para criar e publicar sites sem precisar saber programar.

Configuração inicial — faça uma vez só

1

Crie uma conta no GitHub

O GitHub é onde os arquivos dos seus sites ficam guardados na nuvem — pense nele como um "Google Drive" para sites. É gratuito.

  • a Acesse github.com e clique em "Sign up"
  • b Crie com seu e-mail e escolha um nome de usuário (ex: joao-silva)
  • c Confirme o e-mail — o plano gratuito já é suficiente
Criar conta no GitHub →
✓ Já tem conta? Pule para o próximo passo.
2

Crie uma conta na Vercel

A Vercel é quem publica o seu site na internet — de graça, em segundos, com link automático. Cada vez que você atualizar o site, ele publica sozinho.

  • a Acesse vercel.com e clique em "Sign Up"
  • b Clique em "Continue with GitHub" — use a mesma conta do passo 1
  • c Escolha o plano Hobby (gratuito) e conclua o cadastro
Criar conta na Vercel →
✓ Já tem conta? Pule para o próximo passo.
3

Instale o Bun

O Bun é o programa que roda o f8 Studio no seu computador. A instalação é feita uma única vez, direto pelo terminal.

💡 No Windows, use o PowerShell — não o Prompt de Comando (CMD).
  • a Pressione Win + X e clique em "Windows PowerShell" ou "Terminal"
  • b Cole o comando abaixo e pressione Enter:
irm bun.sh/install.ps1 | iex
  • c Aguarde terminar. Quando aparecer "Bun was installed successfully", está pronto
  • d Feche e reabra o PowerShell para o Bun ser reconhecido
💡 No Mac, use o Terminal — pressione ⌘ + Espaço, escreva "Terminal" e abra.
  • a Abra o Terminal e cole o comando abaixo:
curl -fsSL https://bun.sh/install | bash
  • b Aguarde terminar. Quando aparecer "Bun was installed successfully", está pronto
  • c Feche e reabra o Terminal para o Bun ser reconhecido
✓ Para confirmar: rode bun --version no terminal — deve aparecer um número de versão.
4

Baixe o f8 Studio para o seu computador

Vamos baixar os arquivos do f8 usando o terminal. Siga cada passo com atenção — copie os comandos exatamente como estão.

Parte 1 — Instalar o Git

O Git é um programa que permite baixar o f8. Precisa instalar uma vez só.

  • 1 Abra o seu navegador (Chrome, Edge, Firefox — qualquer um) e acesse:
    git-scm.com/download/win ↗
  • 2 O download do instalador vai começar automaticamente. Se não começar, clique em "Click here to download"
  • 3 Quando terminar o download, abra a pasta Downloads e dê dois cliques no arquivo Git-...-64-bit.exe
  • 4 Uma janela de instalação vai abrir. Clique em "Next" em todas as telas sem mudar nada — as opções padrão já estão certas. No final clique em "Install" e depois em "Finish"

Parte 2 — Abrir o PowerShell

  • 5 Pressione a tecla Win (tecla do Windows no teclado) para abrir o menu Iniciar
  • 6 Digite powershell com o teclado — vai aparecer "Windows PowerShell" nos resultados
  • 7 Clique em "Windows PowerShell" para abrir — uma janela escura com texto branco vai aparecer

Parte 3 — Baixar o f8

  • 8 Na janela do PowerShell, copie e cole o comando abaixo e pressione Enter:
    (isso vai mover o terminal para a sua Área de Trabalho)
cd "$env:USERPROFILE\Desktop"
  • 9 Agora cole o comando abaixo e pressione Enter — isso vai baixar todos os arquivos do f8:
git clone https://github.com/8linksapp-maker/f8.git
  • 10 Aguarde terminar. Quando aparecer "done" ou o cursor piscar de novo, está pronto
  • 11 Uma pasta chamada f8 vai ter aparecido na sua Área de Trabalho — você pode fechar o PowerShell agora

Parte 1 — Abrir o Terminal

  • 1 Pressione as teclas ⌘ e Espaço ao mesmo tempo — a busca do Mac vai abrir
  • 2 Digite terminal e pressione Enter — uma janela escura com texto vai aparecer

Parte 2 — Baixar o f8

  • 3 No Terminal, copie e cole o comando abaixo e pressione Enter:
    (isso vai mover o terminal para a sua Área de Trabalho)
cd ~/Desktop
  • 4 Agora cole o comando abaixo e pressione Enter — isso vai baixar todos os arquivos do f8:
git clone https://github.com/8linksapp-maker/f8.git
  • 5 Se aparecer uma janela pedindo para "Instalar Ferramentas de Linha de Comando", clique em "Instalar" e aguarde terminar. Depois rode o comando git clone acima de novo
  • 6 Aguarde terminar. Quando aparecer "done" ou o cursor piscar de novo, está pronto
  • 7 Uma pasta chamada f8 vai ter aparecido na sua Área de Trabalho — você pode fechar o Terminal agora
✓ Pronto quando a pasta f8 aparecer na sua Área de Trabalho.
5

Baixe e instale o Cursor

O Cursor é o editor com inteligência artificial que vai construir as páginas do site automaticamente. Você não precisa escrever nenhum código — o Cursor faz tudo.

  • a Abra o navegador e acesse cursor.com/download ↗
  • b Clique em "Download for Windows" — um arquivo .exe vai ser baixado
  • c Abra a pasta Downloads, dê dois cliques no arquivo Cursor Setup...exe e aguarde instalar
  • d O Cursor vai abrir sozinho ao terminar. Se pedir para criar conta, crie uma conta gratuita
  • e Clique em File → Open Folder, encontre a pasta f8 que você baixou na Área de Trabalho e clique em "Selecionar pasta"
  • f Para abrir o terminal dentro do Cursor, pressione Ctrl + ` (a tecla do acento grave, ao lado do número 1) — uma janela preta vai aparecer na parte de baixo
  • g No terminal, digite bun run dev e pressione Enter — isso inicia o f8 Studio
  • a Abra o navegador e acesse cursor.com/download ↗
  • b Clique em "Download for Mac" — um arquivo .dmg vai ser baixado
  • c Abra a pasta Downloads no Finder e dê dois cliques no arquivo Cursor...dmg
  • d Uma janela vai abrir mostrando o ícone do Cursor e a pasta Aplicativos — arraste o ícone do Cursor para a pasta Aplicativos
  • e Abra o Cursor pela pasta Aplicativos. Se aparecer aviso de segurança, clique em "Abrir mesmo assim"
  • f Se pedir para criar conta, crie uma conta gratuita
  • g Clique em File → Open Folder, encontre a pasta f8 que você baixou na Área de Trabalho e clique em "Open"
  • h Para abrir o terminal dentro do Cursor, pressione ⌘ + ` (a tecla do acento grave, ao lado do número 1) — uma janela preta vai aparecer na parte de baixo
  • i No terminal, digite bun run dev e pressione Enter — isso inicia o f8 Studio
✓ Pronto quando o terminal mostrar que o f8 está rodando e você conseguir abrir localhost:4321 no navegador.

6

Configure o token do GitHub

O token é uma senha especial que permite ao f8 criar repositórios no GitHub automaticamente. É criado em menos de 1 minuto.

💡 Faça isso com o f8 Studio aberto no navegador em localhost:4321 — você vai colar o token lá no final.
  • a Clique neste link: Gerar token no GitHub ↗
  • b No campo "Note" escreva: f8-studio
  • c Em "Expiration" selecione "No expiration"
  • d Confirme que a caixa repo está marcada e clique em "Generate token"
  • e Copie o token agora (começa com ghp_) — ele não aparece de novo
  • f No f8 Studio, clique em "Configurações" na sidebar, cole o token e salve
🔒 O token fica salvo só no seu computador. Nunca é enviado para servidores externos.

Configuração concluída!

Agora você pode criar seu primeiro site. Abra o f8 Studio e clique em "Criar Site".

Como criar um site — do zero ao ar

1
Abra o f8 Studio
No CursorAntigravity, com a pasta f8 aberta, rode bun run dev no terminal e acesse localhost:4321 no navegador.
2
Crie um novo site
Clique em "Criar Site" na barra lateral, dê um nome ao projeto (ex: minha-clinica) e clique em Criar.
3
Cole a URL do site de referência
Abra o site que você quer copiar no navegador, copie o endereço da barra de cima e cole no f8. Pode colocar várias páginas, uma por linha.
4
Gere os prompts
Clique em "Capturar páginas", aguarde terminar e depois clique em "Gerar prompts". Tudo automático.
5
Salve e copie o comando
Clique em "Salvar prompts" e depois em "Copiar comando".
6
Cole no Cursor e aguarde
No Cursor, pressione Ctrl+L⌘L para abrir o chat. Selecione o modo Agent, cole o comando e pressione Enter. Aguarde sem mexer em nada — pode levar alguns minutos.
6
Cole no Antigravity e aguarde
No Antigravity, abra o chat de IA (Ctrl+L⌘L). Selecione o modo Agent, cole o comando e pressione Enter. Aguarde sem mexer em nada — pode levar alguns minutos.
7
Publique na Vercel
De volta ao f8 Studio, vá em "Meus Sites", clique em "Publicar na Vercel" e siga as instruções. Em minutos o site estará no ar com um link público.

Dúvidas frequentes

Preciso saber programar?
Não. O editor de IA e o f8 fazem todo o trabalho técnico. Você só precisa seguir os passos deste guia.
Quanto custa?
GitHub, Vercel e a conta básica do Cursor/Antigravity são gratuitos. Para uso intenso, planos pagos ficam em torno de R$100/mês.
Quanto tempo leva para criar um site?
A configuração inicial leva ~15 minutos (uma única vez). Depois, criar um site do zero leva entre 30 minutos e 2 horas dependendo da quantidade de páginas.
O site some se eu fechar o computador?
Não. Depois de publicado na Vercel, o site fica no ar 24h por dia, mesmo com seu computador desligado.
Posso usar qualquer site como referência?
Sim, qualquer URL funciona. Recomendamos começar com sites simples de 3 a 5 páginas para o primeiro projeto.
E se o editor de IA travar ou der erro?
Feche o chat, reabra com Ctrl+L⌘L e cole o mesmo comando novamente. O editor consegue continuar de onde parou na maioria dos casos.
E se o f8 Studio não abrir?
Abra o terminal do editor e rode: bun run kill-ports e depois bun run dev. Se ainda não funcionar, feche e reabra o editor.

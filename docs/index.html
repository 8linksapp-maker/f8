Como usar o f8 Studio

Guia passo a passo para criar e publicar sites sem precisar saber programar.

Configuração inicial — faça uma vez só

1

Crie uma conta no GitHub

O GitHub é onde os arquivos dos seus sites ficam guardados na nuvem — pense nele como um "Google Drive" para sites. É gratuito.

  • a Acesse github.com e clique em "Sign up"
  • b Crie com seu e-mail e escolha um nome de usuário (ex: joao-silva)
  • c Confirme o e-mail — o plano gratuito já é suficiente
Criar conta no GitHub →
✓ Já tem conta? Pule para o próximo passo.
2

Crie uma conta na Vercel

A Vercel é quem publica o seu site na internet — de graça, em segundos, com link automático. Cada vez que você atualizar o site, ele publica sozinho.

  • a Acesse vercel.com e clique em "Sign Up"
  • b Clique em "Continue with GitHub" — use a mesma conta do passo 1
  • c Escolha o plano Hobby (gratuito) e conclua o cadastro
Criar conta na Vercel →
✓ Já tem conta? Pule para o próximo passo.
3

Baixe e instale o Cursor

O Cursor é o editor com inteligência artificial que vai construir as páginas do site automaticamente. Você não precisa escrever nenhum código — o Cursor faz tudo.

  • a Acesse cursor.com e clique em "Download"
  • b Instale normalmente — Windows: execute o .exe / Mac: arraste para Aplicativos
  • c Crie uma conta gratuita no Cursor quando ele pedir
  • d Abra a pasta f8 no Cursor: File → Open Folder
  • e No terminal do Cursor, rode bun run dev — isso inicia o f8 Studio
Baixar o Cursor →
✓ Já tem o Cursor? Pule para o próximo passo.
4

Configure o token do GitHub

O token é uma senha especial que permite ao f8 criar repositórios no GitHub automaticamente. É criado em menos de 1 minuto.

💡 Faça isso com o f8 Studio aberto no navegador em localhost:4321 — você vai colar o token lá no final.
  • a Clique neste link: Gerar token no GitHub ↗
  • b No campo "Note" escreva: f8-studio
  • c Em "Expiration" selecione "No expiration"
  • d Confirme que a caixa repo está marcada e clique em "Generate token"
  • e Copie o token agora (começa com ghp_) — ele não aparece de novo
  • f No f8 Studio, clique em "Configurações" na sidebar, cole o token e salve
🔒 O token fica salvo só no seu computador. Nunca é enviado para servidores externos.
🎉

Configuração concluída!

Agora você pode criar seu primeiro site. Abra o f8 Studio e clique em "Criar Site".

Como criar um site — do zero ao ar

1
Abra o f8 Studio
No Cursor, rode bun run dev no terminal e acesse localhost:4321 no navegador.
2
Crie um novo site
Clique em "Criar Site" na barra lateral, dê um nome ao projeto (ex: minha-clinica) e clique em Criar.
3
Cole a URL do site de referência
Abra o site que você quer copiar no navegador, copie o endereço da barra de cima e cole no f8. Pode colocar várias páginas, uma por linha.
4
Gere os prompts
Clique em "Capturar páginas", aguarde terminar e depois clique em "Gerar prompts". Tudo automático.
5
Salve e copie o comando
Clique em "Salvar prompts" e depois em "Copiar comando".
6
Cole no Cursor e aguarde
No Cursor, pressione Ctrl+L (Windows) ou ⌘L (Mac). Selecione o modo Agent, cole o comando e pressione Enter. Aguarde sem mexer em nada — pode levar alguns minutos.
7
Publique na Vercel
De volta ao f8 Studio, vá em "Meus Sites", clique em "Publicar na Vercel" e siga as instruções. Em minutos o site estará no ar com um link público.

Dúvidas frequentes

Preciso saber programar?
Não. O Cursor e o f8 fazem todo o trabalho técnico. Você só precisa seguir os passos deste guia.
Quanto custa?
GitHub, Vercel e a conta básica do Cursor são gratuitos. O Cursor tem um limite de uso no plano gratuito — para uso intenso, o plano pago custa cerca de R$100/mês.
Quanto tempo leva para criar um site?
A configuração inicial leva ~15 minutos (uma única vez). Depois, criar um site do zero leva entre 30 minutos e 2 horas dependendo da quantidade de páginas.
O site some se eu fechar o computador?
Não. Depois de publicado na Vercel, o site fica no ar 24h por dia, mesmo com seu computador desligado.
Posso usar qualquer site como referência?
Sim, qualquer URL funciona. Recomendamos começar com sites simples de 3 a 5 páginas para o primeiro projeto.
E se o Cursor travar ou der erro?
Feche o chat, reabra com Ctrl+L (ou ⌘L) e cole o mesmo comando novamente. O Cursor consegue continuar de onde parou.
E se o f8 Studio não abrir?
Abra o terminal do Cursor e rode: bun run kill-ports e depois bun run dev. Se ainda não funcionar, feche e reabra o Cursor.

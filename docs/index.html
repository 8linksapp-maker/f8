Como usar o f8 Studio

Guia passo a passo para criar e publicar sites sem precisar saber programar.

⚠️ Antes de começar: selecione seu sistema e editor abaixo — o guia vai se adaptar para você

Configuração inicial — faça uma vez só

1

Crie uma conta no GitHub

O GitHub é onde os arquivos dos seus sites ficam guardados na nuvem — pense nele como um "Google Drive" para sites. É gratuito.

  • a Abra o navegador e acesse github.com, clique em "Sign up"
  • b Digite seu e-mail, crie uma senha e escolha um nome de usuário (ex: joao-silva) — use só letras, números e hífen
  • c Confirme seu e-mail clicando no link que o GitHub vai te enviar — sem isso a conta não ativa
  • d Quando perguntar qual plano, escolha "Free" — o gratuito já é suficiente
Criar conta no GitHub →
✓ Já tem conta? Pule para o próximo passo.
2

Crie uma conta na Vercel

A Vercel é quem publica o seu site na internet — de graça, em segundos, com link automático. Cada vez que você atualizar o site, ele publica sozinho.

  • a Acesse vercel.com e clique em "Sign Up"
  • b Clique em "Continue with GitHub" — use a mesma conta que você criou no passo 1
  • c O GitHub vai pedir para autorizar a Vercel — clique em "Authorize Vercel"
  • d Quando perguntar sobre o plano, escolha Hobby (gratuito) e conclua o cadastro
Criar conta na Vercel →
✓ Já tem conta? Pule para o próximo passo.
3

Instale o Bun

O Bun é o programa que roda o f8 Studio no seu computador. A instalação é feita uma única vez pelo terminal.

Parte 1 — Abrir o PowerShell

💡 Use o PowerShell — não o "Prompt de Comando" (CMD). São programas diferentes e o CMD não vai funcionar.
  • 1 Pressione a tecla Win no teclado (ícone do Windows) para abrir o menu Iniciar
  • 2 Digite powershell — vai aparecer "Windows PowerShell" nos resultados. Clique nele
  • 3 Uma janela azul ou preta com texto branco vai abrir — isso é o PowerShell

Parte 2 — Instalar o Bun

⚠️ Se aparecer um aviso do Windows perguntando "Deseja permitir que este aplicativo faça alterações?" — clique em Sim. É normal e seguro.
  • 4 Clique uma vez dentro da janela do PowerShell para garantir que está selecionado
  • 5 Clique no botão "Copiar" ao lado do comando abaixo, depois clique com o botão direito do mouse dentro do PowerShell para colar (no PowerShell, Ctrl+V nem sempre funciona):
irm bun.sh/install.ps1 | iex
  • 6 Pressione Enter e aguarde. Vão aparecer várias linhas de texto — isso é normal
  • 7 Se aparecer um erro sobre "scripts desativados" ou "ExecutionPolicy", cole este comando e pressione Enter para liberar, depois repita o comando do passo 5:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
  • 8 Quando aparecer "Bun was installed successfully", está pronto. Feche o PowerShell — isso é necessário para o Bun ser reconhecido

Parte 1 — Abrir o Terminal

  • 1 Pressione ⌘ e Espaço ao mesmo tempo — a busca do Mac (Spotlight) vai abrir
  • 2 Digite terminal e pressione Enter — uma janela preta ou branca com texto vai aparecer

Parte 2 — Instalar o Bun

  • 3 Clique no botão "Copiar" ao lado do comando abaixo, cole no Terminal com ⌘+V e pressione Enter:
curl -fsSL https://bun.sh/install | bash
  • 4 Se o Mac pedir para instalar as "Ferramentas de Linha de Comando", clique em "Instalar" e aguarde. Depois rode o comando acima de novo
  • 5 Vão aparecer várias linhas de texto — isso é normal. Aguarde terminar
  • 6 Quando aparecer "Bun was installed successfully", está pronto. Feche o Terminal — isso é necessário para o Bun ser reconhecido
✓ Para confirmar que funcionou: reabra o terminal, digite bun --version e pressione Enter — deve aparecer um número como 1.x.x.
4

Baixe o f8 Studio para o seu computador

Vamos baixar os arquivos do f8 usando o terminal. Siga cada passo com atenção.

Parte 1 — Instalar o Git

O Git é necessário para baixar o f8. Precisa instalar uma vez só.

  • 1 Abra o navegador e acesse git-scm.com/download/win ↗
  • 2 O download vai começar automaticamente. Se não começar, clique em "Click here to download"
  • 3 Abra a pasta Downloads e dê dois cliques no arquivo Git-...-64-bit.exe
  • 4 Se aparecer o aviso "Deseja permitir alterações?", clique em Sim
  • 5 Clique em "Next" em todas as telas sem mudar nada. No final clique em "Install" e depois em "Finish"

Parte 2 — Baixar o f8

  • 6 Abra um novo PowerShell (tecla Win → digite "powershell" → clique nele)
  • 7 Cole o comando abaixo com o botão direito e pressione Enter — isso vai ir para a sua Área de Trabalho:
cd "$env:USERPROFILE\Desktop"
  • 8 Agora cole este comando e pressione Enter — vai baixar todos os arquivos do f8:
git clone https://github.com/8linksapp-maker/f8.git
  • 9 Vão aparecer linhas como "Cloning into 'f8'..." e uma barra de progresso. Aguarde terminar
  • 10 Quando o cursor piscar de novo (sem mostrar mais texto), está pronto. Uma pasta chamada f8 apareceu na sua Área de Trabalho
  • 11 Pode fechar o PowerShell agora

Parte 1 — Abrir o Terminal

  • 1 Pressione ⌘ + Espaço, digite terminal e pressione Enter

Parte 2 — Baixar o f8

  • 2 Cole o comando abaixo com ⌘+V e pressione Enter — isso vai ir para a sua Área de Trabalho:
cd ~/Desktop
  • 3 Agora cole este comando e pressione Enter — vai baixar todos os arquivos do f8:

  • 5 Vão aparecer linhas como "Cloning into 'f8'..." e uma barra de progresso. Aguarde terminar
  • 6 Quando o cursor piscar de novo (sem mostrar mais texto), está pronto. Uma pasta chamada f8 apareceu na sua Área de Trabalho
  • 7 Pode fechar o Terminal agora
✓ Pronto quando a pasta f8 aparecer na sua Área de Trabalho.

Instale o Cursor e abra o f8

O Cursor é o editor com IA que vai construir o site. Você não precisa escrever nenhum código.

Parte 1 — Instalar o Cursor

  • 1 Acesse cursor.com/download ↗ e clique em "Download for Windows"
  • 2 Abra a pasta Downloads, dê dois cliques no arquivo CursorSetup...exe
  • 3 Se aparecer o aviso "Deseja permitir alterações?", clique em Sim
  • 4 Aguarde instalar — o Cursor vai abrir sozinho ao terminar. Crie uma conta gratuita quando pedir

Parte 2 — Abrir a pasta f8 no Cursor

  • 5 No Cursor, clique em File no menu superior e depois em Open Folder
  • 6 Na janela que abrir, navegue até a Área de Trabalho e clique na pasta f8 uma vez para selecioná-la, depois clique em "Selecionar pasta"
  • 7 Se aparecer uma mensagem "Do you trust the authors?", clique em "Yes, I trust"

Parte 3 — Instalar as dependências e iniciar o f8

  • 8 Abra o terminal dentro do Cursor: pressione Ctrl + ` (tecla do acento grave, ao lado do número 1). Uma janela preta vai aparecer na parte de baixo
  • 9 Cole o comando abaixo com o botão direito e pressione Enter — isso instala os arquivos necessários (só precisa fazer uma vez):
bun install
  • 10 Aguarde terminar. Quando o cursor piscar de novo, cole o próximo comando e pressione Enter — isso inicia o f8 Studio:
bun run dev
  • 11 Aguarde aparecer o endereço http://localhost:4321 no terminal. Abra o navegador e acesse esse endereço — o f8 Studio vai abrir

Configure o token do GitHub

O token é uma senha especial que permite ao f8 criar repositórios no GitHub automaticamente. É criado em menos de 1 minuto.

💡 Faça isso com o f8 Studio já aberto no navegador em localhost:4321 — você vai colar o token lá no final.
  • a Clique neste link para ir direto para a página de criação do token: Gerar token no GitHub ↗
  • b O GitHub pode pedir para confirmar sua senha — entre com a senha da sua conta
  • c O campo "Note" já vai estar preenchido com f8-studio — não mude
  • d Em "Expiration", clique no menu e selecione "No expiration"
  • e Role a página para baixo e confirme que a caixa repo está marcada com ✓
  • f Clique no botão verde "Generate token" no final da página
  • g Copie o token agora — ele começa com ghp_ e aparece em verde. Se sair da página, ele some para sempre e você precisará criar outro
  • h Vá para o f8 Studio no navegador, clique em "Configurações" na barra lateral, cole o token no campo indicado e clique em Salvar
🔒 O token fica salvo só no seu computador. Nunca é enviado para servidores externos.

Como criar um site — do zero ao ar

1
Abra o f8 Studio
Abra o CursorAntigravity com a pasta f8 aberta, rode bun run dev no terminal e acesse localhost:4321 no navegador.
2
Crie um novo site
Clique em "Criar Site" na barra lateral, dê um nome ao projeto (só letras minúsculas e hífen, ex: minha-clinica) e clique em Criar.
3
Cole a URL do site de referência
Abra o site que você quer copiar no navegador, copie o endereço completo da barra de endereços (começa com https://) e cole no f8. Pode colocar várias páginas, uma por linha.
4
Capture e gere os prompts
Clique em "Capturar páginas" e aguarde — pode levar alguns minutos. Depois clique em "Gerar prompts". Tudo automático.
5
Salve e copie o comando
Clique em "Salvar prompts" e depois em "Copiar comando".
6
Cole no Cursor e aguarde
No Cursor, pressione Ctrl+L⌘+L para abrir o chat. No menu do chat, troque de "Chat" para "Agent". Cole o comando (Ctrl+V⌘+V) e pressione Enter. Não mexa em nada enquanto o Cursor trabalha — pode levar de 5 a 20 minutos.
6
Cole no Antigravity e aguarde
No Antigravity, abra o chat de IA (Ctrl+L⌘+L). Selecione o modo Agent, cole o comando e pressione Enter. Não mexa em nada enquanto trabalha — pode levar de 5 a 20 minutos.
7
Publique na Vercel
De volta ao f8 Studio (localhost:4321), vá em "Meus Sites", clique em "Publicar na Vercel" e siga as instruções. Em minutos o site estará no ar com um link público que você pode compartilhar.

Dúvidas frequentes

Preciso saber programar?
Não. O editor de IA e o f8 fazem todo o trabalho técnico. Você só precisa seguir os passos deste guia.
Quanto custa?
GitHub, Vercel e a conta básica do Cursor/Antigravity são gratuitos. Para uso intenso, planos pagos ficam em torno de R$100/mês.
Quanto tempo leva para criar um site?
A configuração inicial leva ~30 minutos (uma única vez). Depois, criar um site do zero leva entre 30 minutos e 2 horas dependendo da quantidade de páginas.
O site some se eu fechar o computador?
Não. Depois de publicado na Vercel, o site fica no ar 24h por dia, mesmo com seu computador desligado. O f8 Studio (localhost:4321) só funciona com o computador ligado — mas isso só é necessário para criar e editar sites, não para mantê-los no ar.
Posso usar qualquer site como referência?
Sim, qualquer URL funciona. Recomendamos começar com sites simples de 3 a 5 páginas para o primeiro projeto.
E se o editor de IA parar no meio ou der erro?
Feche o chat, reabra com Ctrl+L⌘+L e cole o mesmo comando novamente. O editor consegue continuar de onde parou na maioria dos casos. Se continuar com erro, vá em "Meus Sites" no f8 e clique em "Publicar" mesmo assim — muitas vezes o site já está pronto.
E se o f8 Studio não abrir no navegador?
Abra o terminal dentro do editor e rode os dois comandos abaixo, um de cada vez:

bun run kill-ports
bun run dev

Se ainda não funcionar, feche e reabra o editor com a pasta f8 aberta e tente de novo.
O bun run dev deu erro de "cannot find package"
Significa que o bun install não foi rodado. Abra o terminal dentro do editor com a pasta f8 aberta e rode:

bun install

Aguarde terminar e depois rode bun run dev normalmente.
